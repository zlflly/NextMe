inline 函数在类内定义时不用写 inline 关键字，写实现时再写 inline



能在**类内直接初始化**的变量只有 const类型，static int const 不行（要用初始化列表）



头文件命名不能用数字开头，不能用-，只能用字母开头和下划线

可以在另外一个头文件当中继续另外一个头文件当中的namespace定义

- 先加载上一个头文件

- 直接

  ```cpp
  namespace your_namespce {
  	// 继续写就行，会自动合并
  }
  ```

如果要用这个namespace，要在cpp文件中，引入完善这个 namespace 最后一个链条的头文件

可以在函数中引入头文件(如without_space_using)，如果需要反复写name的话可以考虑，否则还是不建议，会增加编译开销



如果类内函数不需要接受参数，就可以直接用这个类内的变量和函数（好像如果接受参数也可以，只要在类内函数内部，就是操作外来参数/变量+操作类内变量）



如果类的声明在 h.hpp,类的定义在 h.cpp,那么如果想使用这个类，那么该怎么加载



**类内**能够直接初始化的只有 const 类型，static int const都不行，要用初始化列表，能用初始化列表

**类外**main函数当中，一般的const变量和引用值在定义时就要初始化
